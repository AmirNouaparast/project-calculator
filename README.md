# Scientific Calculator

A simple **scientific calculator** built with **HTML / CSS / JavaScript (vanilla)**.  
It comes with a dark UI, numeric and operator buttons, and basic scientific functions (sin, cos, tan, log, factorial, sqrt, exponent, etc.).
